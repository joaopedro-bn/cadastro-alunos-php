# Cadastro de usuários

Esse sistema foi criado para colocar em prática meus estudos em Banco de Dados e a integração do banco com o PHP via **PDO**.

## Tecnologias Utilizadas
* **PHP 8** (PDO para conexão segura)
* **MySQL** (Banco de dados relacional)
* **Bootstrap 5** (Interface responsiva)
* **Arquitetura MVC** (Organização de pastas)

## Como Instalar e Rodar
1. Clone este repositório.
2. Certifique-se de ter o **XAMPP** ou servidor similar instalado.
3. Importe o banco de dados:
   - Abra o `phpMyAdmin`.
   - Crie um novo banco de dados chamado `alunos`.
   - Vá em **Importar** e selecione o arquivo que está em `/database/alunos.sql`.
4. Inicie o servidor e acesse via `localhost`.
